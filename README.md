## Medical - сайт для частной клиники
Проект созданный в рамках учебной практики, с использованием **Laravel и MySql**.

В приложении реализованны 3 роли пользователей: Администратор, Доктор, Клиент.
* **Администратор** ответственнен за наполнение сайта информацмией. Создание, изменение и удаление:
    * Специальностей и специалистов
    * Категорий услуг и услуг
* **Клиент** может сделать запись на прием к специалисту, а также просматривать свои записи и посещения
* **Доктор** просматривает записи на прием и при приеме клиента имеет возможность прикрепить результат приема(исследования) к записи
* **Неавторизованный пользователь** просматривает страницы с информацией о клинике.
